# 项目命名体系

## 系统层（技术平台）
**走马村「云脉寿岭·荔水走马」AIGC 云脑系统**

## 建筑空间层（设计课程作业）
**织境·走马 — 乡村四境空间的数字织造与场景营造**

### 命名逻辑
- **织** = 将分散的人、树、园、道、院编织成可感知的服务网络
- **境** = 四境空间模型（古道叙事境 / 荔田共生境 / 韧谷研学境 / 岭上共居境）
- **数字织造** = AIGC + IoT + 数据流对物理空间的数字建构
- **场景营造** = 不做景区复制，以真实乡村资源为核心的场景化设计

## 可能的分层扩展
- 技术实现层：云脉走马 — AIGC云脑系统设计与实现
- 运营策略层：待定

**Why:** 用户需要为建筑设计课程作业确定标题，"织境·走马"准确对应四境空间模型和编织式的设计手法。

**How to apply:** 在建筑层面相关文档、展板、答辩材料中使用"织境·走马"作为主标题，"乡村四境空间的数字织造与场景营造"作为副标题。
