# 桌面 PowerShell 一键启动设计

## 目标

在当前 Windows 用户的桌面创建“走马村云脑系统”快捷方式，用户双击后可直接启动项目。

## 实现

- 通过 Windows Shell 创建标准 `.lnk` 快捷方式，不复制项目脚本。
- 快捷方式目标为系统 PowerShell。
- 参数为 `-NoExit -ExecutionPolicy Bypass -File "D:\1\AIGC\start.ps1"`。
- 工作目录固定为 `D:\1\AIGC`。
- 桌面路径由 Windows API 获取，兼容桌面目录重定向。

## 行为与错误处理

- PowerShell 窗口保持打开，启动进度和失败信息均可见。
- 项目环境检查、Docker、数据库迁移和前后台启动继续由现有 `start.ps1` 负责。
- 如果项目目录或脚本不存在，创建过程立即失败，不生成无效快捷方式。

## 验证

- 检查快捷方式文件存在。
- 读取快捷方式并核对目标、参数和工作目录。
- 不在验证阶段真正启动服务，避免占用 3000/3001 端口。
