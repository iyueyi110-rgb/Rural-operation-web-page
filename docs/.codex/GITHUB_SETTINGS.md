## Social Preview 上传步骤

1. 文件位置：`docs/assets/social-preview.png`
2. 规格要求：`1200×630px`（已验证为 1200×630）
3. 上传路径：GitHub Settings → General → Social preview → Upload an image
4. 验证方法：匿名模式访问仓库首页，检查预览图显示、裁切和免责声明是否清晰

如果图片不存在，跳过上传并将其列入待办；当前图片已存在，可直接上传。
