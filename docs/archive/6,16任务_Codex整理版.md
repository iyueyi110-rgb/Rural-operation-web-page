## 一、项目总目标

以“村庄云脑”为核心，从三类人群出发，整合村庄资源、生产、生态、游客行为与运营数据，形成服务村庄运营和游客参与的智能系统。

- 服务对象：村民、游客（外来人员）、运营工作人员。
- 游客参与方式：认养、研学、节气活动、互动体验，重点突出“认养一棵树”。
- 村民参与方式：生产管理、农产品反馈、任务协同、运营建议和收益查看。
- 运营方式：监测、分析、反馈、调度和持续优化。
- 空间建设原则：只围绕核心资源和系统节点开展低干预改造。

## 二、核心定位

### 2.1 云脑定位

1. 以村庄云脑为核心。
2. 整合村庄资源与运营数据。
3. 服务村庄运营。
4. 支撑游客参与。
5. 从资源出发组织空间与功能。

### 2.2 核心服务人群

| 人群 | 主要诉求 |
| --- | --- |
| 村民 | 生产管理、任务接收、经营支持、收益反馈 |
| 运营人员 | 数据看板、活动组织、资源调度、问题识别和决策反馈 |
| 游客（外来） | 认养、研学、活动、文旅体验、农产品消费 |

### 2.3 核心资源

- 原生态示范区。
- 水脉与河岸生态。
- 荔枝及多生态果园。
- 古道文化。
- 山地、生态公园和自然景观。
- 闲置房屋、院落与民宿资源。
- 研学工坊。

## 八、空间改造原则

### SPACE-01 基本原则

- 只围绕核心资源展开。
- 只围绕系统节点展开。
- 不做过多无关建设。
- 采用点装置入和低干预方式。
- 保持原生态特征。
- 改造闲置建筑和院落。
- 保留可生长性，允许后续迭代扩展。

### SPACE-02 示范节点

- 果树认养点。
- 农产品展示点。
- 河岸生态研学点。
- 村民服务点。
- 活动集散点。

- 生态资源没有得到有效开发。

- 使用物理设备采集数据。
