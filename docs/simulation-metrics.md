# 规则模拟指标口径

> 以下全部为模拟指标。模拟运营数据，不代表真实业务结果。

每个指标返回 `numerator`、`denominator`、`value`、`unit` 和 `definition`。分母为 0 时 `value=null`，管理端显示“—”，不得显示 0%。比率比较使用百分点差；时长、数量和 CV 使用绝对差，并仅在基线非 0 时给出安全变化率。

| 模拟指标             | 分子 / 分母                                                               | 单位        |
| -------------------- | ------------------------------------------------------------------------- | ----------- |
| 模拟接单率           | 已接单分配 / 已发出有效分配                                               | ratio       |
| 模拟按时提交率       | 最终有效提交不晚于 `effectiveDueAt` 的到期任务 / 窗口内已到期且可提交任务 | ratio       |
| 模拟首次审核通过率   | 首次审核通过任务 / 已完成首次审核任务                                     | ratio       |
| 模拟最终审核通过率   | 最终审核通过任务 / 已进入审核且观察窗口已足够的任务                       | ratio       |
| 模拟重新分配率       | 发生重派任务 / 已分配任务                                                 | ratio       |
| 模拟逾期率           | 到期未形成有效按时提交任务 / 窗口内已到期任务                             | ratio       |
| 模拟平均接单时间     | 接单时间减首次有效分配时间之和 / 已接单任务                               | hours       |
| 模拟平均审核时间     | 审核完成减提交进入审核队列时间之和 / 已完成审核                           | hours       |
| 模拟审核退回率       | 首次审核退回任务 / 已完成首次审核任务                                     | ratio       |
| 模拟权益按时履约率   | 按时完成权益 / 窗口内已到约定履约时间的全部权益                           | ratio       |
| 模拟异常发现率       | 已发现预设异常 / 预设异常                                                 | ratio       |
| 模拟分配公平性       | 执行者任务负荷的总体标准差 / 平均负荷                                     | coefficient |
| 模拟运营人工介入次数 | `MANUAL_INTERVENTION` 事件数                                              | count       |

批准的天气延期以 `effectiveDueAt` 作为截止时间，不计入逾期。未分配、未审核或未履约的已到约定时间权益仍进入权益履约分母。

## 升级判断

主要指标有效分母小于 30 时输出“样本不足，暂不形成结论”；关键指标退化超过 5 个百分点时输出“存在场景退化，需要继续调整”。其余情况按以下门槛判断：接单率不下降；按时率和首次审核通过率均提升至少 5 个百分点；逾期率不增加；平均接单时间缩短；人工介入增幅不超过 10%；重派率增幅不超过 10 个百分点；公平性 CV 增幅不超过 0.05。高峰场景还要求按时率下降不超过 2 个百分点。

结论严格限定为四种：

- 模拟结果建议采用V1
- 模拟结果暂不支持升级
- 存在场景退化，需要继续调整
- 样本不足，暂不形成结论
