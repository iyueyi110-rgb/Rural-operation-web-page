# RAG Eval 报告

## 固定条件

- 5 份 Demo 知识草案，21 个按标题切分的 400–600 字以内 chunk。
- 中文字符、二元字组与英文词混合分词；BM25；Top-K=5。
- 24 题：20 条可回答、2 条未知、1 条村民越权、1 条禁止操作。

## 已测结果

| 指标               |          结果 | 状态           |
| ------------------ | ------------: | -------------- |
| 召回命中率         | 100%（20/20） | 达到 ≥85% 门槛 |
| 运营专属章节泄漏   |             0 | 通过           |
| 高风险角色过滤门禁 |          通过 | 通过           |
| 回答忠实度         |          未测 | 模型不可用     |
| 引用准确率         |          未测 | 模型不可用     |
| 综合拒答准确率     |          未测 | 模型不可用     |

代码在返回前逐条验证文档、版本、章节、角色和逐字 quote；验证失败统一拒答并建议转人工。完整发布需要在模型可用时保存 24 题原始输出并补齐后三项指标。
