# 安全与使用边界

本仓库是私有项目，面向授权的产品评审、开发和内部演示。未经权利人书面许可，不得再分发、公开镜像、商业使用或将仓库中的数据、截图、规则材料用于外部项目。

## 报告安全问题

请通过私有渠道向项目维护者报告安全问题，避免在公开 Issue、Pull Request 或演示材料中披露可利用细节。报告应包含受影响文件或组件、复现条件、影响范围和建议的临时缓解措施。维护者会在确认收到后安排复核，并在修复或缓解完成后同步处理结果。

## 数据与凭证边界

- 不要提交 `.env.local`、访问令牌、云密钥、真实手机号、精确坐标、真实订单或可识别个人的信息。
- 运行产物、Notebook 虚拟环境、缓存、原始截图和本地日志均属于开发环境文件，不能作为发布附件。
- `outputs/simulation/` 中的数据是模拟运营数据，不代表真实业务结果，也不能据此作出生产决策。

## 许可决策

当前不添加 `LICENSE`、开源 package 元数据或开源声明。未来如需发布，必须先完成权属确认、第三方依赖审查和单独的许可证决策。
